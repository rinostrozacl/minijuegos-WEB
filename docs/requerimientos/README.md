# Requerimientos: Plataforma de Competencia de Minijuegos

Este directorio contiene la documentación de requerimientos funcionales y no funcionales para la plataforma web de la competencia de minijuegos con temáticas chilenas.

## Documentos en esta Carpeta

- [**REQUERIMIENTOS_GENERALES.md**](./REQUERIMIENTOS_GENERALES.md): Introducción, alcance, descripción general y listado completo de requerimientos funcionales y no funcionales.
- [**CASOS_USO.md**](./CASOS_USO.md): Descripción detallada de los casos de uso principales del sistema.

## Documentación Relacionada

### Módulos Funcionales
La documentación detallada de cada módulo se encuentra en la carpeta [/docs/modulos/](/docs/modulos/):

- **AUTENTICACION.md**: Módulo de autenticación, registro y verificación de correo.
- **REGISTRO_PARTICIPANTES.md**: Módulo de registro de participantes y selección de temáticas.
- **GESTION_JUEGOS.md**: Módulo de gestión y visualización de juegos.
- **EVALUACION.md**: Módulo de evaluación de juegos.
- **NOTIFICACIONES.md**: Módulo de notificaciones por correo.
- **ADMINISTRACION.md**: Módulo de administración y preregistro masivo.

### Implementación Técnica
Los documentos relacionados con la implementación técnica se encuentran en la carpeta [/docs/implementacion/](/docs/implementacion/):

- **ARQUITECTURA.md**: Arquitectura técnica, componentes principales y modelo de datos.
- **PLAN_IMPLEMENTACION.md**: Fases de desarrollo, cronograma y consideraciones para la implementación.
- **CREDENCIALES_FIREBASE.md**: Configuración de Firebase y gestión de credenciales.
- **GUIA_IMPLEMENTACION.md**: Guía detallada para la implementación de la plataforma.
