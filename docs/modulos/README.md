# Módulos Funcionales: Plataforma de Competencia de Minijuegos

Este directorio contiene la documentación detallada de cada módulo funcional para la plataforma web de la competencia de minijuegos con temáticas chilenas.

## Documentos en esta Carpeta

- [**AUTENTICACION.md**](./AUTENTICACION.md): Módulo de autenticación, registro y verificación de correo mediante código.
- [**REGISTRO_PARTICIPANTES.md**](./REGISTRO_PARTICIPANTES.md): Módulo de registro de participantes y selección de temáticas.
- [**GESTION_JUEGOS.md**](./GESTION_JUEGOS.md): Módulo de gestión y visualización de juegos WebGL.
- [**EVALUACION.md**](./EVALUACION.md): Módulo de evaluación de juegos según criterios específicos.
- [**NOTIFICACIONES.md**](./NOTIFICACIONES.md): Módulo de notificaciones por correo electrónico.
- [**ADMINISTRACION.md**](./ADMINISTRACION.md): Módulo de administración y preregistro masivo de alumnos.
