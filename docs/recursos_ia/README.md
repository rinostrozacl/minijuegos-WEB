# Recursos para Generación con IA

Este directorio contiene recursos específicos para facilitar la generación de código mediante Inteligencia Artificial para la plataforma de competencia de minijuegos con temáticas chilenas.

## Documentos en esta Carpeta

- [**PLANTILLAS_CODIGO.md**](./PLANTILLAS_CODIGO.md): Plantillas y ejemplos de código para los componentes principales.
- [**GUIA_ESTILOS.md**](./GUIA_ESTILOS.md): Convenciones de nomenclatura, estructura de archivos y patrones a seguir.
- [**DIAGRAMA_DEPENDENCIAS.md**](./DIAGRAMA_DEPENDENCIAS.md): Visualización de cómo los diferentes módulos se relacionan entre sí.
- [**CASOS_PRUEBA.md**](./CASOS_PRUEBA.md): Ejemplos de casos de prueba para validar la funcionalidad implementada.
- [**PROMPTS_IA.md**](./PROMPTS_IA.md): Ejemplos de prompts efectivos para solicitar a la IA la generación de componentes específicos.
- [**REGLAS_SEGURIDAD.md**](./REGLAS_SEGURIDAD.md): Definiciones de las reglas de seguridad para Firestore y Storage.
- [**FLUJOS_INTEGRACION.md**](./FLUJOS_INTEGRACION.md): Documentación sobre la integración de servicios externos.

## Cómo Utilizar estos Recursos

Estos recursos están diseñados para ser utilizados en conjunto con la documentación principal del proyecto (requerimientos, implementación y módulos) para facilitar la generación de código mediante IA.

1. Consulta primero los requerimientos y la arquitectura para entender el contexto general.
2. Utiliza las plantillas de código como base para la generación.
3. Sigue la guía de estilos para mantener la consistencia.
4. Verifica las dependencias entre módulos antes de generar código nuevo.
5. Utiliza los prompts sugeridos como punto de partida para tus solicitudes a la IA.
6. Implementa las reglas de seguridad según lo especificado.
7. Sigue los flujos de integración para conectar con servicios externos.
