# Competencia de Minijuegos - Temáticas Chilenas

## Descripción del Proyecto

Este proyecto contiene recursos y código para desarrollar una competencia de minijuegos basados en temáticas chilenas, incluyendo su fauna, flora, humedales, mitología y pueblos originarios.

## Estructura del Proyecto

- `tematicas_chilenas.md`: Contiene 72 ideas de temáticas para videojuegos centradas en Chile.
- `tipos_de_juegos.md`: Clasificación y descripción de 19 categorías de juegos extraídos de las temáticas.

## Objetivos

1. Desarrollar minijuegos educativos y entretenidos basados en la cultura y biodiversidad chilena.
2. Crear un sistema de competencia donde los jugadores puedan participar en diferentes minijuegos.
3. Implementar un sistema de puntuación y clasificación para los participantes.
4. Promover el conocimiento sobre la naturaleza y cultura de Chile a través de experiencias interactivas.

## Próximos Pasos

- Seleccionar temáticas específicas para desarrollar.
- Diseñar la arquitectura del sistema de minijuegos.
- Implementar prototipos de los primeros minijuegos.
- Desarrollar el sistema de competencia y puntuación.

## Recursos

Todos los datos y conversaciones relacionadas con este proyecto se almacenan en la base de datos vectorial `dataauto.db` ubicada en la raíz del proyecto.
