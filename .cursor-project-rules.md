## Resumen rápido

- **Stack:** Nuxt 3 + Nitro + Firebase (Auth, Firestore, Storage) + Resend
- **Despliegue:** GitHub → Coolify; DNS Cloudflare
- **Juego =** documento `themes` reservado; build en **GitHub Pages** (`gameWebGLUrl`)
- **Admin canónico:** `allowed-emails` con `type: admin` (código aún inconsistente — ver guía)
- **Contexto IA:** carpeta `.agents/` + `docs/` (no `.ia/`)

## Antes de codear

1. Leer `.agents/guia-agentica.md` y `docs/README.md`
2. Cambios mínimos; convenciones del repo existente
3. Tras cambios grandes: actualizar `historial.md` y docs afectados

## MCP Firebase

Configurado en `~/.cursor/mcp.json` del desarrollador. Ver `docs/mcp-firebase.md` para herramientas y precauciones.
