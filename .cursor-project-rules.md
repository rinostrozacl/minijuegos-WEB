## Resumen rápido

- **Stack:** Nuxt 3 + Nitro + Firebase (Auth, Firestore, Storage) + Resend
- **Despliegue:** GitHub → Coolify; DNS Cloudflare (no usar docker-compose del repo como fuente de prod)
- **Juego =** documento `themes` reservado; build WebGL en `public/games/{themeId}/`
- **Admin canónico:** `allowed-emails` con `type: admin` (código aún inconsistente — ver guía)
- **Contexto IA:** carpeta `.agents/` (no `.ia/`)

## Antes de codear

1. Leer `.agents/guia-agentica.md`
2. Cambios mínimos; convenciones del repo existente
3. Tras cambios grandes: actualizar `.agents/context/historial.md`
